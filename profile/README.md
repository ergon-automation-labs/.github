Ergon Automation Labs
=====================

**Personal AI that respects your attention.**

Bot Army is a distributed operating system of 18+ specialist agents on Elixir/OTP and NATS — designed for decision support, not decision automation. Push surfaces interrupt only when it matters. Pull surfaces wait until you're ready.

**Works with your tools:** Connect Claude Desktop or Cursor via MCP. Create tasks, query your knowledge graph, roll dice, get chronicle briefs — all from your AI assistant.

Get started in 60 seconds:

```
curl -fsSL https://raw.githubusercontent.com/ergon-automation-labs/ergon-starter/main/install.sh | bash
```

**What's public:** Bot runtimes, skills platform, starter packs, MCP integration, and the job search stack. The private system is where the full 18-bot fleet runs; these repos are the pieces worth sharing.

**Philosophy:** Collapse the option space so you can act confidently under time pressure.
